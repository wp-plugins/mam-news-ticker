Mam News Ticker is an awesome, super lightweight plugin for your wordpress website. 

== Description ==

This plugin will enable news ticker in your wordpress theme. You can embed news ticker via shortcode in everywhere you want, even in theme files. 

Wanna see how it works? Click here: http://mamunkhan.net/plugins/mam-news-ticker/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want news ticker in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates

Shortcodes

<strong>General Ticker</strong>
<pre>ticker id="1"]</pre>


<strong>Ticker form category</strong>
<pre>ticker id="2" category="1"]</pre>


<strong>Speed change</strong>
<pre>ticker id="3" speed="1000"]</pre>

 
<strong>TypeSpeed Change</strong>
<pre>ticker id="4" typespeed="100"]</pre>


<strong>Color variation</strong>
<pre>ticker id="5" color="red"]</pre>

 
<strong>Text change!</strong>
<pre>ticker id="6" text="Breaking News!"]</pre>
